#### Cheat Sheet
Install: 
```bash
cd ~/.config
mkdir LSD
cd LSD
curl https://raw.githubusercontent.com/nico-Zero/config/main/.config/LSD/cht.sh > cht.sh
curl https://raw.githubusercontent.com/nico-Zero/config/main/.config/LSD/.cht_language_list.txt > .cht_language_list.txt
```
